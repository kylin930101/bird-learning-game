# 🐦 鸟类认知学习游戏 - 使用说明

## 📖 游戏简介

这是一个专为鸟类爱好者设计的学习游戏，包含13种常见鸟类的认知学习。游戏由**小林** 🐱 为**姐姐**精心制作，旨在通过有趣的游戏方式帮助用户学习鸟类知识。

## 🎮 游戏特色

### 1. 学习模式 📚
- 浏览所有13种鸟类的精美图片
- 查看每种鸟类的详细信息：
  - 中文名称
  - 学名（拉丁文）
  - 主要特征描述
  - 栖息地信息
- 适合初学者系统学习

### 2. 识别模式 🔍
- 看图猜鸟名挑战
- 每次显示一张鸟类图片
- 从4个选项中选择正确答案
- 共10道题目，每题10分
- 答题后显示正确答案和详细解释

### 3. 记忆模式 🧠
- 经典翻牌配对游戏
- 12张牌（6对鸟类图片）
- 60秒时间限制
- 锻炼记忆力和反应速度
- 每配对成功一对得20分

### 4. 挑战模式 🏆
- 限时答题挑战
- 60秒内回答尽可能多的问题
- 每题15分，答错不扣分
- 实时进度条显示
- 争夺最高分排行榜

## 🚀 如何启动游戏

### 方法一：使用启动脚本（推荐）
1. 双击 `启动游戏.bat` 文件
2. 等待Python服务器启动
3. 浏览器会自动打开游戏页面
4. 如果浏览器没有自动打开，请手动访问：`http://localhost:8000`

### 方法二：使用Python命令
1. 打开命令行（CMD或PowerShell）
2. 切换到游戏目录：
   ```bash
   cd "D:\games\bird-quiz-web"
   ```
3. 运行服务器：
   ```bash
   python start-server.py
   ```
4. 在浏览器中访问：`http://localhost:8000`

### 方法三：直接打开HTML文件
1. 双击 `index.html` 文件
2. 游戏会在默认浏览器中打开
3. **注意**：某些功能（如图片加载）可能受限，建议使用方法一或二

## 📱 多设备访问

### 电脑访问
- 任何现代浏览器都支持（Chrome、Firefox、Edge、Safari等）
- 推荐使用Chrome或Edge以获得最佳体验

### 手机/平板访问
1. **确保手机和电脑在同一WiFi网络**
2. 查看电脑的IP地址：
   - Windows: 在命令行输入 `ipconfig`，查找"IPv4地址"
   - Mac: 在终端输入 `ifconfig`，查找"inet"
3. 在手机浏览器中输入：`http://[电脑IP地址]:8000`
   - 例如：`http://192.168.1.100:8000`
4. 现在可以在手机上玩游戏了！

### 平板电脑
- 与手机访问方式相同
- 大屏幕更适合学习模式和记忆模式

## 🎯 游戏技巧

### 学习建议
1. **先学习后测试**：建议先使用"学习模式"熟悉所有鸟类
2. **循序渐进**：从识别模式开始，逐步挑战记忆和挑战模式
3. **重复练习**：多次游戏有助于加深记忆
4. **观察细节**：注意鸟类的嘴型、羽毛颜色、体型等特征

### 高分技巧
1. **识别模式**：仔细对比选项，注意鸟类学名
2. **记忆模式**：先记住鸟类位置，再快速配对
3. **挑战模式**：保持冷静，快速但不草率地选择答案

## 🔧 常见问题

### Q1: 游戏无法启动
- **检查Python是否安装**：在命令行输入 `python --version`
- **端口被占用**：尝试修改 `start-server.py` 中的 `PORT` 变量为其他值（如8080）
- **权限问题**：尝试以管理员身份运行脚本

### Q2: 图片无法显示
- **检查文件路径**：确保 `birds` 文件夹中有所有13张图片
- **文件名正确**：图片文件名必须与代码中的名称完全一致
- **使用服务器**：建议使用本地服务器启动，而不是直接打开HTML文件

### Q3: 手机无法访问
- **检查网络**：确保手机和电脑在同一WiFi网络
- **防火墙设置**：临时关闭电脑防火墙或添加端口8000的例外
- **IP地址正确**：使用 `ipconfig` 查看正确的IP地址

### Q4: 游戏运行缓慢
- **浏览器缓存**：清除浏览器缓存后重试
- **图片大小**：所有图片已优化，但首次加载可能需要时间
- **网络问题**：确保网络连接稳定
