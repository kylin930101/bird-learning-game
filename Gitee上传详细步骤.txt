# 🐱 Gitee上传文件详细步骤（给姐姐的教程）

## 第一步：登录Gitee
1. 打开浏览器，访问：https://gitee.com
2. 登录你的账号

## 第二步：进入仓库
1. 在浏览器地址栏输入你的仓库地址：
   https://gitee.com/你的用户名/bird-quiz-web
2. 按回车进入

## 第三步：找到上传按钮
在仓库页面，仔细找这些地方：

### 位置A：文件列表上方
```
[代码] [Issues] [Pull Requests] [Wiki] [统计] [设置]
↑ 确保在"代码"标签页

在文件列表区域上方，你会看到：
┌─────────────────────────────────────────────┐
│ 搜索文件...                                 │
│ [新建文件] [上传文件] [克隆/下载] ▼         │
└─────────────────────────────────────────────┘
↑ 点击"上传文件"按钮
```

### 位置B：如果没有看到"上传文件"
1. 点击"新建文件"按钮
2. 在新页面，看顶部或底部是否有"上传文件"选项
3. 或者：在文件列表空白处右键，可能有上传选项

## 第四步：上传文件
1. 点击"上传文件"后，会出现上传区域
2. 两种方式：
   - 方式1：把电脑里的文件直接拖到网页的虚线框里
   - 方式2：点击"选择文件"，从电脑选择文件

## 第五步：选择要上传的文件
在你的电脑上找到：D:/games/bird-quiz-web/

### 先上传这些测试（建议顺序）：
1. 📄 gitee-index.html    （主页面）
2. 📄 game.js             （游戏逻辑）
3. 📄 style.css           （样式文件）
4. 📄 birds-data.js       （鸟类数据）
5. 📄 404.html            （错误页面）

### 再上传文件夹：
6. 📁 images/             （鸟类图片文件夹）
   - 注意：上传文件夹时，Gitee可能需要你逐个选择文件
   - 或者：把整个images文件夹拖进去

## 第六步：提交文件
上传完成后，在页面最底部：
```
提交更改
┌─────────────────────────────────────────────┐
│ 提交信息：                                   │
│ [Initial commit: 鸟类知识问答游戏]          │
│                                             │
│ 提交说明：（可选）                           │
│ [首次提交，包含核心游戏文件]                │
│                                             │
│ [✓] 直接提交到master分支                    │
│                                             │
│ [提交更改]                                  │
└─────────────────────────────────────────────┘
```

1. 在"提交信息"输入：Initial commit: 鸟类知识问答游戏
2. 勾选"直接提交到master分支"
3. 点击"提交更改"

## 第七步：等待上传完成
1. 页面会刷新
2. 在文件列表中看到你上传的文件
3. 如果有绿色提示"提交成功"，就完成了！

## 🆘 常见问题解决

### 问题1：找不到"上传文件"按钮
- 刷新页面
- 换个浏览器试试（Chrome、Edge）
- 确保在"代码"标签页，不是"Issues"或"Wiki"

### 问题2：上传失败
- 检查网络连接
- 文件不要太大（单个文件<100MB）
- 文件名不要有特殊字符

### 问题3：图片文件夹上传麻烦
如果images文件夹上传困难：
1. 先上传几个核心图片测试
2. 或者用压缩包上传：
   - 把images文件夹压缩成zip
   - 上传zip文件
   - Gitee会自动解压（部分版本支持）

### 问题4：上传后文件不见了
- 刷新页面
- 检查是否在正确的分支（master）
- 查看"提交历史"确认上传成功

## 🎮 上传后的测试
1. 开启Gitee Pages（在仓库"服务"→"Gitee Pages"）
2. 等待1-3分钟部署
3. 访问：https://你的用户名.gitee.io/bird-quiz-web/gitee-index.html
4. 测试游戏是否正常运行
