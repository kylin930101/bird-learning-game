## 🚀 部署步骤

### 第1步：注册Gitee账号
1. 访问 [gitee.com](https://gitee.com)
2. 点击右上角"注册"
3. 填写邮箱、用户名、密码
4. 完成邮箱验证

### 第2步：创建仓库
1. 登录后点击右上角"+" → "新建仓库"
2. 填写仓库信息：
   - **仓库名称**：`bird-quiz-web`（推荐）
   - **仓库介绍**：鸟类认知学习游戏 - 通过游戏学习鸟类知识
   - **是否开源**：选择"公开"
   - **初始化仓库**：**不要**勾选任何选项
   - **设置模板**：选择"不选择"
3. 点击"创建"

### 第3步：上传文件
#### 方法一：Web界面上传（推荐）
1. 进入刚创建的仓库
2. 点击"上传文件"按钮
3. 将`bird-quiz-web`文件夹中的所有文件拖拽到上传区域
4. 填写提交信息："初始提交：鸟类认知学习游戏"
5. 点击"提交"

#### 方法二：Git命令上传
```bash
# 1. 初始化Git仓库
git init

# 2. 添加远程仓库
git remote add origin https://gitee.com/你的用户名/bird-quiz-web.git

# 3. 添加所有文件
git add .

# 4. 提交更改
git commit -m "初始提交：鸟类认知学习游戏"

# 5. 推送到Gitee
git push -u origin master
```

### 第4步：开启Gitee Pages
1. 进入仓库页面
2. 点击上方菜单栏的"服务" → "Gitee Pages"
3. 在Pages设置页面：
   - **部署分支**：选择`master`
   - **部署目录**：选择`/`（根目录）
   - **强制使用HTTPS**：勾选
4. 点击"启动"按钮
5. 等待部署完成（约1-2分钟）

### 第5步：访问网站
部署完成后，你会看到：
- **访问地址**：`https://你的用户名.gitee.io/bird-quiz-web`
- **示例地址**：`https://xiaolin.gitee.io/bird-quiz-web`

## 🔧 高级配置

### 自定义域名
1. 准备一个已备案的域名
2. 在域名服务商处添加CNAME记录：
   ```
   记录类型：CNAME
   主机记录：www（或@）
   记录值：你的用户名.gitee.io
   ```
3. 在Gitee Pages设置中填写自定义域名
4. 等待DNS生效（约10-30分钟）

### 自动部署
每次向仓库推送代码时，Gitee Pages会自动重新部署。

### 监控统计
在`.gitee-pages.yml`中配置统计代码：
```yaml
monitoring:
  baidu_analytics: "你的百度统计ID"
  google_analytics: "你的谷歌分析ID"
```

## ⚠️ 常见问题

### Q1：页面显示空白
**原因**：文件路径错误或缺少资源
**解决**：
1. 检查控制台错误信息
2. 确认所有文件已上传
3. 检查`.gitee-pages.yml`配置

### Q2：图片无法加载
**原因**：图片路径错误或格式不支持
**解决**：
1. 检查`images/`文件夹是否上传
2. 确认图片格式为`.jpg`、`.png`或`.webp`
3. 检查CSS中的图片路径

### Q3：部署失败
**原因**：配置文件错误或超时
**解决**：
1. 检查`.gitee-pages.yml`语法
2. 删除`.gitee-pages.yml`重新部署
3. 联系Gitee客服

### Q4：访问速度慢
**原因**：图片过大或网络问题
**解决**：
1. 优化图片大小
2. 启用Gitee Pages的CDN缓存
3. 使用国内DNS
