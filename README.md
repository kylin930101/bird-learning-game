# 🐦 鸟类认知学习游戏

欢迎来到**鸟类认知学习游戏**！这是一款帮助大家认识各种鸟类的趣味学习游戏，由小林和姐姐共同制作。🐱

## 🎮 游戏特色

- **精美界面**：森林主题设计，色彩鲜艳，视觉舒适
- **丰富内容**：包含10种常见鸟类，配有详细描述和精美图片
- **互动学习**：通过选择题形式，边玩边学鸟类知识
- **多关卡挑战**：随着关卡提升，题目数量和难度增加
- **音效反馈**：正确答案有鼓励音效，错误答案有提示音效
- **键盘支持**：支持键盘快捷键操作

## 🚀 快速开始

1. **直接打开**：双击 `index.html` 文件即可在浏览器中运行游戏
2. **在线预览**：上传到服务器后通过链接访问

## 🎯 游戏玩法

### 基本规则
1. 观察鸟类图片，从4个选项中选择正确的鸟名
2. 答对得10分，答错不得分
3. 每完成10题进入下一关卡
4. 游戏时间60秒，时间到游戏结束

### 控制方式
- **鼠标点击**：选择答案、点击按钮
- **键盘快捷键**：
  - `1-4`：选择对应选项
  - `空格`：下一题
  - `H`：显示提示
  - `R`：重新开始游戏

### 游戏界面
- **顶部信息栏**：显示得分、关卡、进度和时间
- **鸟类展示区**：显示鸟类图片和描述
- **选择题区**：4个选项按钮
- **反馈区**：显示答题结果和提示
- **知识卡片**：鸟类相关知识介绍
- **游戏说明**：详细玩法说明

## 🔧 自定义扩展

### 添加新鸟类
在 `game.js` 文件的 `birds` 数组中添加新鸟类：

```javascript
{
    id: 11,
    name: "新鸟类名称",
    image: "图片URL",
    description: "鸟类描述",
    options: ["正确名称", "错误选项1", "错误选项2", "错误选项3"]
}
```

### 修改游戏参数
- 初始时间：修改 `timeLeft` 初始值
- 每题分数：修改 `score` 增加量
- 关卡题目数：修改 `totalQuestions` 初始值和增加逻辑
