# 鸟类学习游戏 🐦

一个基于Web的鸟类学习游戏，通过图片识别鸟类并学习相关知识，类似birdquiz.my的功能。

## 🌟 功能特点

### 🎮 **游戏模式**
- 通过鸟类图片识别鸟类名称
- 10道题目挑战
- 计时器和积分系统
- 即时反馈和鸟类知识学习

### 📚 **学习模式**
- 详细的鸟类图鉴
- 每种鸟类的详细信息（名称、学名、描述、栖息地、食性等）
- 学习进度跟踪

### 🏆 **成就系统**
- 最高分记录
- 已学习鸟类统计
- 正确率统计
- 游戏历史记录

### 🎨 **精美界面**
- 响应式设计，适配手机、平板和电脑
- 现代化的UI设计
- 流畅的动画效果
- 深色模式支持

### ⚙️ **个性化设置**
- 难度等级选择（简单、中等、困难）
- 声音开关
- 数据导入导出
- 统计信息查看

## 🐦 鸟类数据

### 当前包含的鸟类（使用真实图片）
游戏包含5种真实鸟类图片（姐姐提供）：
1. **鹊鸭** - 中型水鸟，黑白相间的羽毛
2. **斑头秋沙鸭** - 小型水鸟，头部有黑白斑纹
3. **普通秋沙鸭** - 大型水鸟，细长的红色喙
4. **红胸秋沙鸭** - 中型水鸟，醒目的红色胸部
5. **环颈雉** - 美丽的雉类，雄性有白色颈环

### 鸟类图片说明
游戏现在使用姐姐提供的真实鸟类图片，图片位于 `birds/` 目录下。

1. **准备图片**：收集鸟类图片（.jpg, .png格式）
2. **放置图片**：放到 `images/birds/` 目录
3. **更新路径**：在 `js/data.js` 中更新图片路径
4. **命名建议**：使用鸟类名称拼音，如 `maque.jpg`, `yanzi.jpg`

### 添加新鸟类
要添加更多鸟类：
1. 在 `js/data.js` 的 `birdsData` 数组中添加新对象
2. 添加图片到 `images/birds/` 目录
3. 确保格式与其他鸟类一致

3. **使用本地服务器（推荐）**
   ```bash
   # 使用Python
   python -m http.server 8000

   # 或使用Node.js
   npx serve .
   ```
   然后在浏览器访问：http://localhost:8000

## 🎯 游戏玩法

### 开始游戏
1. 点击"开始游戏"按钮
2. 查看显示的鸟类图片
3. 从4个选项中选择正确的鸟类名称
4. 查看反馈信息，学习鸟类相关知识
5. 完成10道题目，争取最高分

### 游戏规则
- **每题时间**：60秒（可配置）
- **每题分数**：10分
- **提示次数**：每局3次（使用会扣2分）
- **难度等级**：
  - 简单：90秒，3个选项
  - 中等：60秒，4个选项
  - 困难：30秒，5个选项

### 键盘快捷键
- **数字键1-4**：选择选项
- **空格键**：跳过当前题目
- **H键**：使用提示
- **ESC键**：返回主菜单

## 🛠️ 扩展开发

### 添加新鸟类
1. 在 `js/data.js` 的 `birdsData` 数组中添加新鸟类数据
2. 将鸟类图片添加到 `images/birds/` 目录
3. 图片会自动生成占位图（如果图片不存在）

### 修改游戏配置
在 `js/data.js` 的 `gameConfig` 对象中修改：
- `totalQuestions`：题目总数
- `timePerQuestion`：每题时间
- `pointsPerCorrectAnswer`：每题分数
- `maxHintsPerGame`：提示次数限制

### 自定义样式
- 修改 `css/style.css`：基础样式和变量
- 修改 `css/game.css`：游戏界面样式
- 修改 `css/responsive.css`：响应式样式

## 🐛 故障排除

### 常见问题
1. **图片无法加载**
   - 检查图片路径是否正确
   - 系统会自动生成占位图片

2. **声音无法播放**
   - 检查浏览器是否允许自动播放
   - 在设置中开启声音

3. **数据无法保存**
   - 检查浏览器是否支持LocalStorage
   - 尝试清除浏览器缓存

4. **界面显示异常**
   - 刷新页面
   - 清除浏览器缓存
   - 检查浏览器兼容性

- **基本支持**：Edge 90+, Opera 76+
