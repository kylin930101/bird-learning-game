📝 使用说明

姐姐，这些是占位图片！你可以：

  1. 保存这些图片：右键点击每个圆形图片 → 另存为
  2. 保存到正确位置：保存到 bird-quiz-web/images/birds/placeholders/ 目录
  3. 命名规则：保存为 bird1.png, bird2.png, ..., bird10.png
  4. 替换真实图片：以后可以用真实的鸟类图片替换这些占位图

或者：把你自己的鸟类图片放到 images/birds/ 目录，然后在 data.js 中更新图片路径！
